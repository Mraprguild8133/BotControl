Bot Commands

Available commands for users and administrators

User Commands

/start Start the bot and get welcome message
/help Show all available commands
/search <movie> Search for movies
/download <movie> Get download links
/getid Get your user and chat ID
/contact Get contact information

Admin Commands

/admin Open admin panel
/addchannel Add channel to management
/addkeyword <keyword> Add copyright filter keyword
/setwelcome <message> Set welcome message
/adminstats View bot statistics
/testai <text> Test AI content detection
